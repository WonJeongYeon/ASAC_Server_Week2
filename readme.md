# ASAC Server 2주차

## 숫자야구 프로그램

### 기초 설정
- [x] 자릿수 : 4
- [ ] 재시작 여부 : X
- [ ] 횟수 제한 : X
- [ ] 횟수 카운팅 : O

### 게임 설정
- [x] 게임 시작 (빌드) 시 무작위 4자리 숫자 생성
- [x] 4 스트라이크 (승리) 시 까지 반복
  - [x] 사용자가 4자리 숫자 입력
  - [x] 스트라이크, 볼 개수 출력
- [ ] 시도한 횟수 및 종료 멘트 출력

### 예외처리
- [x] 사용자가 입력한 값이 숫자인가?
- [x] 사용자가 입력한 값이 4자리인가?
- [ ] 사용자는 서로 다른 값을 입력했는가?
